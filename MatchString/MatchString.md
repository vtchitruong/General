Determine how many times a string occurs in the list.  

### Constraints
1 <= n <= 1000  
1 <= n <= 1000  
1 <= |strings[i]|, |queries[i]| <= 20  

### Sample Input
4  
aba  
baba  
aba  
xzxb  
3  
aba  
xzxb  
ab  

### Sample Output
2 1 0

### Explanation
"aba" occurs twice -> 2  
"xzxb" occurs once -> 1  
"ab" does not occur -> 0
